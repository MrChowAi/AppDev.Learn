# Git Commands I Actually Use

## Every Day
- `git pull` - Get latest from GitHub
- `git status` - See what changed
- `git add .` - Stage everything
- `git commit -m "message"` - Save snapshot
- `git push` - Send to GitHub

## Sometimes
- `git branch` - See my branches
- `git checkout -b new` - Create and switch to new branch
- `git merge branch-name` - Combine branches

## Mistakes
- `git restore <file>` - Undo changes to a file
- `git reset HEAD~1` - Undo last commit (keep changes)

## GitHub
- `git clone <url>` - Copy someone's repo to my computer
- `git remote -v` - Check if connected to GitHub
